# 🚀 DoctorFizz – SEO Keyword & Content Intelligence Platform

DoctorFizz is a **Next.js-based SEO intelligence and content optimization platform** that helps users analyze websites, discover keyword opportunities, generate SEO insights, and build data-driven content using modern APIs and AI-powered pipelines.

---

## ✨ Features

- 🌐 Website SEO analysis & onboarding flow
- 🌍 Language & location–based SEO targeting
- 🔑 Keyword discovery & competitor analysis
- 📊 SEO opportunities & site profiling
- ✍️ AI-powered content research (Perplexity integration)
- 🧠 Modular API architecture (Next.js App Router)
- 🎨 Modern UI with Tailwind CSS
- 🔐 Secure environment-based configuration

---

## 🛠 Tech Stack

- **Framework:** Next.js (App Router)
- **Frontend:** React, Tailwind CSS
- **Backend:** Next.js API Routes
- **AI / SEO Providers:**  
  - Perplexity API  
  - Google APIs (GA4, GSC – optional)
- **Tooling:** ESLint, PostCSS
- **Package Manager:** npm
